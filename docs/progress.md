## Steps
- Completing 3 projects allows to apply for jobs in technology (JavaScript/TypeScript, PHP, Python, GO, Java, C#, C, C++).
- English fluency allows to apply for global jobs (USA, CA, UK, etc).
- Learn and practice DSA allows to apply for jobs in big techs (Google, Microsoft, Amazon, etc).
